# 简历画像分析器原型说明

项目当前回到一个简单的产品入口：左侧输入年龄和简历，右侧输出 DeepSeek 生成的简短简历洞察。

核心入口：

- 页面：`index.html`
- 后端：`server.js`
- API 客户端：`ai-client.js`
- 前端交互：`app.js`
- 样式：`styles.css`
- 启动文档：`README.md`

本版本重点验证：

- 根据年龄和简历丰富度，输出同龄进展评分
- 支持粘贴文本，也支持上传 TXT、MD、PDF、DOCX 简历
- 识别 3 个关键能力领域，并说明可迁移使用场景
- 判断 3 个适合工作方向，每个方向一句话解释
- 简述潜在短板
- 简述最应该补齐的能力和经历
- 完成报告后，支持基于简历和前置分析继续流式追问
- 画像只要求结构化文本，不要求 DeepSeek 返回图像
- DeepSeek key 只存在服务端环境变量中

启动方式见 `README.md`。
